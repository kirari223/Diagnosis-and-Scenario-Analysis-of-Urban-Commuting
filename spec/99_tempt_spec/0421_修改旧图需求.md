## 5. 四格局人流箱线图优化（展现极限格局特征）

**输入文件**: `results/3.Situation_Diagnosis/3.1Holistic_Diagnosis/3.1.3Pattern_Comparison/star_格局对比_人流箱线图_四格局_v2.png`

**输出文件**: `star_格局对比_人流箱线图_四格局_v3.png`

**问题**: 当前 `cap=200` 截尾导致极限格局（中位数100+）被严重截断

**方案**: 双子图（用户确认）
- 左图：展示0-200区间（放大细节），cap=200
- 右图：展示全范围（对数轴），不截尾或高截尾（cap=2000）

**修改位置**: `notebooks/sub_pipeline/evaluation.ipynb` 中 `eval_V98_flow_boxplot()` 函数

**实现要点**:
```python
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(24, 7))

# 左图：0-200细节
cap_left = 200.0
data_list_left = [df['人数'][df['人数'] <= cap_left].values for df in dfs]
ax1.boxplot(data_list_left, ...)
ax1.set_xlabel('OD对通勤人数（人）', fontsize=24)
ax1.set_title('细节视图（0-200人）', fontsize=20)

# 右图：对数轴全貌
cap_right = 2000.0  # 或不截尾
data_list_right = [df['人数'][df['人数'] <= cap_right].values for df in dfs]
ax2.boxplot(data_list_right, ...)
ax2.set_xscale('log')
ax2.set_xlabel('OD对通勤人数（人，对数轴）', fontsize=24)
ax2.set_title('全貌视图（对数轴）', fontsize=20)
```
