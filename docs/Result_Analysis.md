### 4.2 Rigidity_Computation — 全量刚性提取

**研究背景**：长沙主城区正经历以数字经济为主导的产业结构转型，马栏山视频文创园、岳麓山大学科技城等新兴产业集聚区的兴起，使得灵活办公、居家办公的制度条件日趋成熟。在此背景下，量化现状通勤格局的职住刚性，是研判未来通勤结构演变的前提。

**方法**：以泊松回归（PPML）对全量通勤 OD 矩阵进行拟合，分别估计 O 端（居住地）和 D 端（就业地）的刚性系数 $\alpha_O$、$\alpha_D$，再通过 $\theta = \frac{\alpha}{1-\alpha} \cdot \frac{1}{\beta}$ 映射为 UOT 模型的惩罚权重。$\beta$ 固定为 Wilson 标定值（米单位 0.0003），作为 offset 引入，不参与估计。

**结果解读**：

$\alpha_O$ 和 $\alpha_D$ 均落在 (0, 1) 区间，反映长沙主城区通勤格局具有显著的职住刚性——居住人口规模和就业岗位规模对通勤流量均有正向拉力，但并非完全刚性（$\alpha < 1$）。两端刚性系数的相对大小揭示了职住约束的不对称性：若 $\alpha_D > \alpha_O$，说明就业端对通勤流的约束更强，居民更倾向于"就业地不变、居住地调整"；反之则说明居住端更稳定。

从城市规划视角看，当前刚性水平是长沙"强省会"战略下就业高度集聚（湘江新区、五一商圈、高新区）与居住空间相对分散之间张力的量化体现。较高的 D 端刚性意味着就业中心的吸引力已形成强烈的向心通勤格局，这与长沙"一江两岸"的空间结构和跨江通勤压力高度吻合。

---

### 4.3 Scenario_Computation — 情景一推演结果

**情景设定**：全量刚性降低 20%（$\theta_O^{\text{scenario}} = \theta_O^{\text{base}} \times 0.8$，$\theta_D^{\text{scenario}} = \theta_D^{\text{base}} \times 0.8$），对应居家办公比例提升、职住约束松动的政策情景。

**模型机制**：UOT 广义 Sinkhorn 迭代在总通勤量守恒（$M = 2{,}212{,}508$ 人）的前提下，允许 $O^*$、$D^*$ 偏离现状锚点，以寻找更低通勤成本的职住匹配。$\theta$ 降低意味着偏离锚点的惩罚减小，模型更倾向于重新分配职住关系。

**结果解读**：

刚性降低 20% 后，情景格局的平均通勤距离相较实际格局有所增加。这一结果初看似乎与"居家办公减少通勤"的直觉相悖，实则揭示了一个重要的城市规划机制：**当职住约束松动时，居民并非简单地"减少通勤"，而是在更大的空间范围内重新优化职住匹配**。部分居民会选择接受更远的通勤距离，以换取更优质的就业岗位或更宜居的居住环境——这正是"弹性通勤"的核心特征。

从空间格局看，$O^*$ 与 $O_0$ 的 KL 散度反映了居住端的调整幅度，$D^*$ 与 $D_0$ 的 KL 散度反映了就业端的调整幅度。若就业端调整幅度更大，说明刚性降低主要通过"换工作"而非"搬家"来实现职住再匹配，这与长沙数字经济岗位流动性强、就业市场活跃的特征相符。

**城市规划含义**：

情景一的推演结果对长沙城市更新政策具有以下启示：

1. **交通基础设施需求不降反升**：刚性降低后通勤距离增加，意味着城市轨道交通、快速路网的覆盖范围需要进一步扩展，尤其是连接湘江两岸和近郊新城的通道。

2. **职住平衡政策的空间靶向**：$O^*$、$D^*$ 偏离现状最显著的 TAZ 是政策干预的优先区域——这些地区的居民或就业者对职住调整最为敏感，是推广混合办公、社区工作站的重点布局区域。

3. **"强省会"战略的通勤代价**：就业高度集聚带来的高 D 端刚性，在弹性化情景下会转化为更大范围的通勤扩散。这提示在推进产业集聚的同时，需同步推进居住配套和职住就近政策，避免通勤成本的系统性上升。

实际格局O端（居住地）和D端（工作地）的TAZ人数分布呈现明显的空间集聚特征。O端高值区集中于城市外围居住组团，D端高值区集中于中心城区就业核心。两端分布的空间错位是通勤需求产生的根本原因。

**职住平衡度（M3/V3）**

全局平衡度均值为4.137，中位数为1.281，分布高度右偏。中位数接近1说明多数TAZ职住基本平衡，但均值偏高反映少数就业密集型TAZ（如商务中心、工业园区）拉高了整体均值。平衡度>2的TAZ主要分布于中心城区核心就业区和外围工业区，平衡度<0.5的TAZ主要分布于大型居住社区。

**街道自给度（M4/V4）**

110个街道中92个有有效数据，自给度均值为0.148，即约14.8%的通勤流在街道内部完成。自给度最高的街道约为0.36，最低约为0.01。整体自给度偏低，说明长沙主城区职住空间分离程度较高，大多数居民需要跨街道通勤。

**标准差椭圆（V92）**

居住地（O端）椭圆与工作地（D端）椭圆的中心位置、长轴方向和覆盖范围均有差异，反映了职住空间分布的方向性特征。两个椭圆的重叠程度可作为职住空间匹配度的直观指标。

### 3.1.3 Pattern_Comparison — 优化潜力层

**超额通勤指标（M4-M7）**

| 指标 | 值 | 含义 |
|------|-----|------|
| C_obs | 5.578 km | 实际平均通勤距离 |
| C_min | 1.118 km | 理论最小通勤距离（理想格局） |
| C_ran | 13.144 km | 随机通勤距离 |
| EC | 79.96% | 实际通勤中约80%属于超额通勤 |
| NEC | 37.08% | 实际格局在理论区间内处于偏低位置 |
| CE | 57.57% | 相比随机格局节约了约58%的通勤距离 |
| NCE | 62.92% | 已实现理论最大节约量的约63% |

NEC+NCE=100%（数学约束验证通过）。NEC=37%说明实际格局距理论最优仍有较大优化空间；CE=58%说明现有职住空间安排相比完全随机已有显著改善。
